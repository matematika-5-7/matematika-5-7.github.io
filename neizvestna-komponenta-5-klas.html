➕➖

Неизвестно събираемо, умаляемо и умалител

3 формули — 3 примера

➕ Неизвестно събираемо: x + b = c → x = c − b
  • Ако x + b = c, то x = c − b (изваждаме известното събираемо от сумата)
  • Пример: x + 3,2 = 7,5 → x = 7,5 − 3,2 = 4,3
➖ Неизвестно умаляемо: x − b = c → x = c + b
  • Ако x − b = c, то x = c + b (прибавяме умалителя към разликата)
  • Пример: x − 1,5 = 4 → x = 4 + 1,5 = 5,5
➖ Неизвестен умалител: a − x = c → x = a − c
  • Ако a − x = c, то x = a − c (изваждаме разликата от умаляемото)
  • Пример: 8 − x = 2,5 → x = 8 − 2,5 = 5,5

✖️➗

Неизвестен множител, делимо и делител

3 формули — 3 примера

✖️ Неизвестен множител: x · b = c → x = c : b
  • Ако x · b = c, то x = c : b (делим произведението на известния множител)
  • Пример: x · 4 = 12,8 → x = 12,8 : 4 = 3,2
➗ Неизвестно делимо: x : b = c → x = c · b
  • Ако x : b = c, то x = c · b (умножаваме частното по делителя)
  • Пример: x : 5 = 3,4 → x = 3,4 · 5 = 17
➗ Неизвестен делител: a : x = c → x = a : c
  • Ако a : x = c, то x = a : c (делим делимото на частното)
  • Пример: 15 : x = 2,5 → x = 15 : 2,5 = 6

Как решаваме по-сложни уравнения?

Пример с обяснение на всяка стъпка

📋 Стратегия за сложни задачи
  1. Пресмятаме всичко, което може да се пресметне (известните части).
  2. Получаваме по-просто уравнение с x.
  3. Прилагаме правилото за съответната компонента.
  4. Правим проверка!
🧩 Пример А — два сбора с x
1
Дадено: x + 2,3 + 1,7 = 12
2
Пресмятаме известното: 2,3 + 1,7 = 4
3
Получаваме: x + 4 = 12
4
Намираме x: x = 12 − 4 = 8
✓
Проверка: 8 + 2,3 + 1,7 = 12 ✅
✓ x = 8
🧩 Пример Б — умножение и събиране
1
Дадено: x · 3 + 4,5 = 19,5
2
Намираме x · 3: x · 3 = 19,5 − 4,5 = 15
3
Намираме x: x = 15 : 3 = 5
✓
Проверка: 5 · 3 + 4,5 = 15 + 4,5 = 19,5 ✅
✓ x = 5
