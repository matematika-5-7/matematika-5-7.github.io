📌 Раздел 1

Събиране и изваждане на десетични дроби

Точно като при обикновените числа — само трябва да наредим запетаите една под друга!

📏

Главното правило: запетаята под запетаята!

3 лесни стъпки

📋 Как се събират/изваждат десетични дроби
  1. Записваме числата едно под друго така, че запетаите да са точно наредени (хилядни под хилядни, стотни под стотни, десетни под десетни).
  2. Числата се събират или изваждат така, както се събират цели числа.
  3. В резултата поставяме запетаята точно под другите запетаи.
💡
Лесен трик! Ако числата имат различен брой цифри след запетаята — допишем нули отдясно, за да ги изравним! Например: 12,8 + 1,4 = 12,8 + 1,4 ✓ но 12,76 − 10,1 → пишем 12,76 − 10,10

Разместително и съдружително свойство При събиране можем да разменяме и групираме числата по удобен начин:
a + b + c = (a + b) + c = a + (b + c)
Пример: 101,3 + 14,23 + 9,7 + 0,77 = (101,3 + 9,7) + (14,23 + 0,77) = 111 + 15 = 126

📌 Раздел 2

Умножение на десетични дроби

Умножаваме като цели числа — после броим цифрите след запетаята!

✖️

Как се умножава?

Правилото е лесно!

📋 Правило за умножение
  1. Умножаваме числата без да гледаме запетаите — като цели числа.
  2. Броим колко цифри има след запетаята в двете числа заедно.
  3. В произведението отброяваме толкова цифри отдясно и поставяме запетаята.
🔍 Примери стъпка по стъпка:
а) 23,1 · 5
1
Умножаваме: 231 · 5 = 1155
2
В 23,1 има 1 цифра след запетаята
3
Отброяваме 1 цифра отдясно
23,1 · 5 = 115,5
б) 0,98 · 2
1
Умножаваме: 98 · 2 = 196
2
В 0,98 има 2 цифри след запетаята
3
Отброяваме 2 цифри: 1,96
0,98 · 2 = 1,96
в) 2,1 · 0,2
1
Умножаваме: 21 · 2 = 42
2
2,1 → 1 цифра; 0,2 → 1 цифра. Общо: 2
3
Отброяваме 2 цифри от 42: 0,42
2,1 · 0,2 = 0,42
🧮
Разпределително свойство — много полезно! Понякога е по-лесно да разбием едно число: (a + b) · c = a · c + b · c
Пример: (5,25 + 4,5) · 0,2 = 5,25 · 0,2 + 4,5 · 0,2 = 1,05 + 0,9 = 1,95
Разместително
a · b = b · a
2,3 · 4,5 = 4,5 · 2,3 = 10,35
Съдружително
(a · b) · c = a · (b · c)
(2,1 · 3) · 0,2 = 2,1 · (3 · 0,2)
Разпределително
(a ± b) · c = a·c ± b·c
(3,56 − 1,23) · 0,2 = 3,56·0,2 − 1,23·0,2

📌 Раздел 3

Деление на десетични дроби

➗

Два вида деление

Десетична : цяло число и Десетична : Десетична

📋 Деление на десетична дроб с цяло число
  1. Делим цифра по цифра отляво надясно — точно като при цели числа.
  2. Когато минем запетаята в делимото → веднага поставяме запетая в частното.
  3. Продължаваме да делим до края.
🇧🇬
Как се записва делението в България — стълбичка! Пише се горният ред с равенството, после под него — всяко изваждане с черта, докато стигнем 0. Запетаята в частното слагаме точно когато я минем в делимото.
📐 Примери — стълбичка (точно така се прави в България):
🔍 Как четем стълбичката:
1. Горният ред е пълното равенство: делимо : делител = частно
2. Всеки ред с „−" показва колко изваждаме
3. Чертата показва края на едно изваждане
4. Под чертата — следващото число за делене
5. Накрая стигаме до 0 — делението е точно!
💡
Кога поставяме запетаята в частното? Точно когато я минем в делимото!
При 13 : 4 — 13 е цяло, делим, стигаме до 1 (остатък), добавяме 0 → 10, тогава слагаме запетая в частното → 3,...
При 0,13 : 4 — запетаята е веднага след 0, затова в частното пишем 0,0...
📋 Деление на десетична дроб с десетична дроб
  1. Преместваме запетаята в делителя докрай вдясно — той трябва да стане цяло число.
  2. С толкова места надясно преместваме запетаята и в делимото.
  3. Делим по вече познатото правило с цяло число — стълбичка!
📐 Примери: Десетична : Десетична → превръщаме в : цяло число
5,13 : 0,3 → местим с 1 → 51,3 : 3
0,28 : 0,007 → местим с 3 → 280 : 7
2,88 : 0,24 → местим с 2 → 288 : 24
✅
Проверка на делението! Винаги можеш да провериш: частно · делител = делимо
Пример: 12 · 0,24 = 2,88 ✓  |  40 · 0,007 = 0,28 ✓
